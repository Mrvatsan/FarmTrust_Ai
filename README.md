# FarmTrust AI - Organic Produce Intelligence System

Build a full-stack React Native + Node.js/Express + TensorFlow.js app for smartphone-based organic/inorganic fruits & vegetables classification.

## Core Features
- YOLOv8/ViT classification with CLAHE & NIR augmentation.
- Nutritional Deficiency Predictor (ResNet-50).
- Recipe Pairing Engine (Mask R-CNN + Transformer).
- Allergy-Risk Scanner (GNN + Pesticide DB).
- Sustainability Carbon Tracker (XGBoost).
- Edge AI Micro-Farm Network (Federated Learning).
- Blockchain Traceability (Ethereum/Solana).

## Project Structure
- `backend/`: Node.js Express server.
- `frontend/`: React Native (Expo) mobile application.
- `models/`: Python scripts for AI/ML modules.
- `contracts/`: Solidity smart contracts for traceability.
